**This script will:**
1.  **Map `.claude/skills`** to this global registry via symbolic links.
2.  **Inject the Indelible Master Protocol** into your project's `CLAUDE.md`.
3.  **Mandate all 12 skills** to be enforced before a single line of code is written.
